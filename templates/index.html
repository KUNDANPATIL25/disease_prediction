About

This Disease Prediction System helps users identify possible diseases based on selected symptoms. It uses machine learning models trained on medical data to provide accurate predictions with probability scores.

Use Guide

  • Select one or more symptoms from the list on the Home section.
  • Use the search box to quickly find symptoms.
  • Click the "Predict" button to get the predicted diseases with their probabilities.
  • Clear your selections anytime using the "Clear" button.
  • Review the prediction results in the popup modal showing multiple possibilities.
